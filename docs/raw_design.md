# Global Game Jam 2026 \- Taipei Team H

今年主辦單位的主題設定為 Mask 由多個有面具意象的影片所構成。

## 概念發想

根據今年主題 Mask（面具、覆蓋、遮罩）的意象，決定設計一款橫向的類塔防遊戲。遊戲玩法遊玩家操作角色，投擲物品（如：彈藥）阻擋從右方進入畫面的敵人。

## 詞彙表

| 單詞 | 說明 |
| :---- | :---- |
| 玩家（Player） | 可操作的角色，能在遊戲畫面中自由移動 |
| 攤位（Booth） | 用來儲存玩家的技能素材，用於後續合成 |
| 食材（Food） | 使用技能的素材，打倒敵人會掉落，敵人也會奪取食物 |
| 餓鬼（Ghost） | 遊戲中的敵人（Enemy） |
| 魔王（Boss） | 回合結束前，會出現特別強大的敵人。 |
| 回合（Wave） | 每一回合會生成不同數量的敵人，越後面的回合敵人越多 |

## 核心玩法

玩家透過普通攻擊打倒敵人，獲取不同的道具素材，在透過合成道具使用強力的技能，盡可能在遊戲中存活下來。

### 攤位系統

玩家可以設置不同的攤位來搜集特定素材，素材可以被用於後續的合成。每個攤位可以保存的道具數量不同。

### 戰鬥系統

玩家可以使用普通攻擊打倒敵人，除此之外可以利用下方的合成槽選擇三個不同的素材合成技能來發動特殊技能。

玩家可以使用普通攻擊，但是有彈夾限制（最大數量 6 個 ）當子彈消耗完畢後需要等待重新填彈的時間（3 秒）

### 合成系統

玩家在遊戲進行中，可以使用搜集到的道具合成使用特殊技能。當選擇三個道具後，會直接消耗素材發動特殊技能。

### 升級系統

每一個回合結束後，可以升級物品，可以透過消耗素材來強化特定的能力。每回合可以選擇一個升級選項，每次只會隨機抽選兩種升級選項。

## 操作方式

玩家主要透過鍵盤進行操作

| 按鍵 | 用途 |
| :---- | :---- |
| W | 向上移動 |
| A | 向左移動 |
| S | 向下移動 |
| D | 向右移動 |
| Space | 進行互動、投擲物品 |
| 1 | 提取第一個道具到合成欄 |
| 2 | 提取第二個道具到合成欄 |
| 3 | 提取第三個道具到合成欄 |

## 遊戲畫面

遊戲畫面由場景跟抬頭顯示兩大部分構成，以 2D 的俯視角（Top-Down）類塔防的方式呈現。畫面以 16:9 的橫向方式呈現。

### 主要場景

左側 20% 空間為攤位區域，垂直方向一共可以設置三個不同的攤位。中間到右側是可活動的區域，在右畫面外會生成敵人，向左移動，碰到攤位區域後會奪取攤位中的資源。

玩家可以在中間到右側的區域自由活動，根據判斷進行對敵人的阻擋。

### 抬頭顯示（HUD）

抬頭顯示器分為畫面上方跟下方兩大區域，上方會顯示本回合餘剩的敵人數量，以及目前存活多少回合。

畫面下方是獨立的顯示畫面，會顯示玩家目前提取的道具準備合成、目前合成後的技能效果。

UI 不屬於遊戲範圍，不能蓋住遊戲畫面影響玩家判斷舞台中的玩家跟怪物。

### 敵人

每個敵人有不同的生命值跟特性，玩家需要盡可能地清怪打倒敵人，打倒敵人時會隨機掉落食材（80%），用於升級子彈。

| 名稱 | 類型 | 血量（Health Point） | 移動速度（Movement Per Second） |
| :---- | :---- | :---- | :---- |
| 餓鬼 | Ghost | 1 | 0.5 |
| 餓死鬼 | Boss | 3 | 0.3 |

### 升級

| 名稱 | 效果 | 消耗 |
| :---- | :---- | :---- |
| 加辣 | 臭豆腐造成的傷害倍率 \+0.5 | 豆腐 x3 |
